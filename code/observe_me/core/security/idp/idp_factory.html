    IDPFactory

    Factory class to instantiate the appropriate Identity Provider adapter.

    This class centralizes the creation logic for IDP adapters, enabling plug-and-play support for different authentication services based on system configuration.

    get_idp 

    get_idp() -> IDPAdapter

    Retrieves and initializes an IDP adapter instance based on environment variables.

    The method reads the IDP_PROVIDER variable to select the implementation class and passes the necessary OIDC parameters (URL, Audience, Issuer) to the constructor.

    Returns:

    Name Type Description
    IDPAdapter IDPAdapter

    An initialized instance of a concrete IDP adapter.

    Raises:

    Type Description
    ValueError

    If the IDP_PROVIDER specified is not supported.
    Environment Variables Used

    IDP_PROVIDER: Type of provider (e.g., 'keycloak', 'logto', 'authentik', 'cognito'). IDP_URL: The JWKS endpoint for token validation. IDP_AUDIENCE: The expected audience claim in the JWT. IDP_ISSUER: The expected issuer claim in the JWT.
