    IDPAdapter

    Bases: ABC

    Abstract Base Class for Identity Provider (IDP) adapters.

    This class manages the logic for fetching public keys from a JWKS endpoint, verifying JWT signatures, and extracting user authorization roles.

    Attributes:

    Name Type Description
    jwks_url str

    URL to the Identity Provider's JWKS endpoint.
    audience str

    The expected 'aud' claim in the JWT.
    issuer str

    The expected 'iss' claim from the IDP.
    algorithms list

    Supported signing algorithms (defaults to ES384).

    get_payload_verified 

    get_payload_verified(token: str) -> dict[str, Any]

    Decodes and validates the token using the IDP's public keys.

    Performs checks for: 1. Signature validity (using remote JWKS). 2. Audience ('aud') match. 3. Issuer ('iss') match.
